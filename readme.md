# Bootstrap Forms

Extension of laravelcollective/html for bootstrap

## Installation

Add the following to composer.json
```json
{
    "require": {
            "artifishall/bootstrap_forms": "dev-master"
        },
    "repositories": [
        {
            "type": "vcs",
            "url": "git@github.com:artifishall/bootstrap_forms.git"
        }
    ]
}
```

Run
```
composer update
```
